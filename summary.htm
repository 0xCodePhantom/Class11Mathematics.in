Chapter 1: Sets

A set is a collection of well-defined objects. Well-defined means every member is clearly identifiable. Examples: • A = {1, 2, 3} • B = {x | x is an even natural number} • Empty Set: φ Types of Sets: • Finite: Countable number of elements. • Infinite: Uncountable (like natural numbers). • Equal Sets: Same elements. • Subset: A ⊆ B means every element of A is in B. Operations on Sets: • Union A ∪ B – all elements from both. • Intersection A ∩ B – common elements. • Difference A − B – elements only in A. • Complement A′ – universal set minus A. Venn Diagrams: Used to visually represent sets. Helps in solving union/intersection problems. Important Results: • n(A ∪ B) = n(A) + n(B) − n(A ∩ B) • For disjoint sets: n(A ∪ B) = n(A) + n(B)

Chapter 2: Relations & Functions

Relation: A relation between two sets A and B is any subset of A × B. Example: If A = {1,2} and B = {3,4}, relation R may be {(1,3), (2,4)}. Function: A relation where each input has exactly one output. Example: f(x) = x². Important Terms: • Domain: All possible inputs. • Codomain: All possible outputs. • Range: Actual outputs. Types of Functions: • One-one (Injective) • Onto (Surjective) • Bijective (both) • Constant function • Identity function

Chapter 3: Trigonometric Functions

Conversion of Degrees & Radians: π radians = 180° Trigonometric Ratios: sin, cos, tan, cot, sec, cosec. Identities: 1. sin²θ + cos²θ = 1 2. 1 + tan²θ = sec²θ 3. 1 + cot²θ = cosec²θ Graphs: • Sine and cosine are periodic with period 2π. • Tangent has period π. Principal Values: Defines the main angle used by inverse trigonometric functions.

Chapter 4: Complex Numbers

Complex number is of form: z = a + ib. Here a = real part, b = imaginary part. Operations: • Addition: (a+ib)+(c+id) • Multiplication • Conjugate: a − ib • Modulus: |z| = √(a² + b²) Argand Plane: Graphical representation of complex numbers.

Chapter 5: Linear Inequalities

Linear inequality is like a linear equation but uses <, >, ≤, ≥. Solving Inequalities: 1. Shift terms normally. 2. Multiply/divide carefully: If both sides multiplied or divided by negative → sign reverses. Graphing Inequalities: • One variable: Show region on number line. • Two variables: Region in plane; boundary line solid or dashed.

Chapter 6: Permutations and Combinations

Permutation: Arrangement where order matters. Formula: P(n, r) = n!/(n−r)! Combination: Selection where order doesn’t matter. Formula: C(n, r) = n!/[r!(n−r)!] Applications: • Arranging digits • Forming committees • Probability

Chapter 7: Binomial Theorem

Expansion of (a + b)^n for natural number n. General Term: T(k+1) = C(n,k) a^(n−k) b^k Middle Term: If n even → 1 middle term If n odd → 2 middle terms

Chapter 8: Sequences and Series

Arithmetic Progression (AP): a, a+d, a+2d... General term: aₙ = a + (n−1)d Sum: Sₙ = n/2[2a + (n−1)d] Geometric Progression (GP): a, ar, ar²... Sum: Sₙ = a(rⁿ−1)/(r−1)

Chapter 9: Straight Lines

Slope: m = (y₂ − y₁)/(x₂ − x₁) Equation Forms: • Slope-intercept: y = mx + c • Two-point form • Intercept form • Normal form Angle Between Lines: tanθ = |(m₁ − m₂)/(1+m₁m₂)|

Chapter 10: Conic Sections

Parabola: y² = 4ax Ellipse: x²/a² + y²/b² = 1 Hyperbola: x²/a² − y²/b² = 1 Each has focus, directrix, eccentricity.

Chapter 11: Introduction to 3D Geometry

Coordinates: (x, y, z) Distance Formula: √[(x₂−x₁)²+(y₂−y₁)²+(z₂−z₁)²] Section Formula: Internal and external division of line segment.

Chapter 12: Limits and Derivatives

Limit: Approach value of function. Examples: lim sinx/x = 1 Derivative: Rate of change. Basic Rules: d/dx x^n = nx^(n−1) d/dx sinx = cosx

Chapter 13: Statistics

Measures: • Mean deviation • Variance • Standard deviation Variance: σ² = Σ(x−mean)² / N

Chapter 14: Probability

Probability = favourable outcomes / total outcomes. Includes sample space, events, complementary events, addition theorem.
