Chapter 8: Sequences and Series

Important Questions with Detailed Answers

Q1. Find the 15th term of the arithmetic progression 7, 10, 13, ...
Formula: an = a + (n−1)d
Here, a = 7, d = 3, n = 15
a15 = 7 + (15−1)×3 = 7 + 42 = 49.
Q2. Find the sum of first 20 terms of the AP: 3, 7, 11, ...
Formula: Sn = n/2 × [2a + (n−1)d]
a = 3, d = 4, n = 20
S20 = 20/2 × [6 + 19×4] = 10 × [6+76] = 10×82 = 820.
Q3. Which term of the AP: 5, 8, 11, ... is 77?
Formula: an = a + (n−1)d
77 = 5 + (n−1)×3 → (n−1)×3 = 72 → n−1 = 24 → n = 25.
Q4. Find the sum of the first 15 terms of the series: 2 + 4 + 8 + ...
This is GP with a=2, r=2, n=15.
Formula: Sn = a(rn−1)/(r−1)
S15 = 2(215−1)/(2−1) = 2(32768−1) = 65534.
Q5. Find the 8th term of the GP: 3, 6, 12, ...
Formula: an = a × rn−1
a = 3, r = 2, n = 8
a8 = 3 × 27 = 3 × 128 = 384.
Q6. Find the sum to infinity of the GP: 5 + 2.5 + 1.25 + ...
Formula: S = a / (1−r) for |r|<1
a = 5, r = 0.5
S = 5 / (1−0.5) = 5 / 0.5 = 10.
Q7. Find the sum of the first 10 multiples of 7.
Multiples: 7, 14, 21, ..., 70 (AP with a=7, d=7, n=10).
S10 = 10/2 × [2×7 + 9×7] = 5 × [14+63] = 5×77 = 385.
Q8. If Sn = n², find the sequence.
Terms: a1=1²=1, a2=S2−S1=4−1=3, a3=9−4=5, ...
Sequence = 1, 3, 5, 7, ... (odd numbers).
Q9. Find the arithmetic mean between 12 and 52.
AM = (a+b)/2 = (12+52)/2 = 64/2 = 32.
Q10. If an = 2n−1, show that the sequence is an AP. Find its 20th term.
Sequence: 1, 3, 5, ... (difference = 2, so AP).
a20 = 2(20)−1 = 40−1 = 39.
