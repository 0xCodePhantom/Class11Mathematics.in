Chapter 5: Linear Inequalities (treated as Chapter 6 originally)

Important questions with clear solutions and diagrams.
Core concepts
Linear inequalities are like linear equations but with inequality signs (<, ≤, >, ≥). Solve them by the same algebraic moves, but remember: when you multiply or divide by a negative number, flip the inequality sign. Solutions are shown on number lines or as intervals.
Q1. Solve and represent on number line: 3x − 5 < 7.
3x − 5 < 7 ⇒ 3x < 12 ⇒ x < 4. On number line: open circle at 4, shade left.
Q2. Solve: −2(3x − 1) ≥ 4.
−2(3x −1) ≥ 4 ⇒ −6x +2 ≥4 ⇒ −6x ≥2 ⇒ x ≤ −1/3 (inequality flips when dividing by −6). Represent as closed circle at −1/3 and shade left.
Q3. Solve: (x − 2)/(x + 3) < 0. Show sign chart and solution.
Critical points: x = 2 (numerator zero), x = −3 (denominator zero). Make sign table: intervals (−∞,−3), (−3,2), (2,∞). Test values: x=−4 ⇒ (−)/(−)=+; x=0 ⇒ (−)/(+)=−; x=3 ⇒ (+)/(+)=+. So inequality <0 on (−3,2). Exclude −3 (undefined), include 2? Numerator zero gives 0 <0 false, so exclude 2. Final: (−3,2).
Q4. Solve and graph: 2x − 3 < x + 5 and x ≥ 0 (system).
First inequality: 2x −3 < x +5 ⇒ x <8. Combine with x ≥0 ⇒ 0 ≤ x < 8. Represent as [0,8) on number line.
Q5. Solve absolute inequality: |x − 2| < 3.
Interpret as −3 < x −2 < 3 ⇒ add 2: −1 < x <5. So interval (−1,5).
Q6. Solve: |2x +1| ≥ 5.
Break into two: 2x +1 ≥5 ⇒ x ≥2; or 2x +1 ≤ −5 ⇒ x ≤ −3. So solution: (−∞,−3] ∪ [2,∞).
Q7. Solve inequality and draw half-planes: x + 2y ≤ 6.
Treat as line x + 2y =6. Rewrite y = (6 − x)/2. Draw line; test origin (0,0): 0 +0 ≤6 true, so shade below line including boundary (solid). This is the feasible half-plane.
Q8. Solve system of inequalities graphically: x ≥ 0, y ≥ 0, x + y ≤ 4.
Graph coordinate axes; line x + y =4 is a straight line from (4,0) to (0,4). Feasible region is the triangle with vertices (0,0),(4,0),(0,4), including boundaries. Show small SVG of triangle.
Q9. Explain solving quadratic inequality: x² − 5x +6 < 0.
Factor: (x −2)(x −3) <0. Roots at 2 and 3. Parabola opens upward. The expression is negative between the roots. So solution: (2,3).
Q10. Word problem: A shop sells packs; at least 10 packs and at most 30 packs must be ordered. Each pack costs ₹45, budget ₹1200. How many packs can be bought?
Let n be number of packs. Conditions: 10 ≤ n ≤ 30 and 45n ≤1200 ⇒ n ≤ 1200/45 = 26.666 ⇒ integer n ≤26. Combine: n ∈ {10,11,...,26}. So maximum 26 packs, minimum 10.
