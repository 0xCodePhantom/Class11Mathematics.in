Chapter 13: Statistics — Important Questions

Measures of central tendency & dispersion — worked examples and visuals.
Quick formula sheet
Mean ( Ungrouped ): \(\bar{x}=\dfrac{\sum x_i}{n}\).
Median: middle value when data sorted (or average of two middle for even n).
Mode: most frequent value.
For grouped data (class intervals): use class marks m_i and frequencies f_i. Mean: \(\bar{x}=\dfrac{\sum f_i m_i}{\sum f_i}\).
Median (grouped): locate median class then interpolate. Mode (grouped): use formula \(\text{Mode} = l + \dfrac{f_m - f_{1}}{2f_m - f_{1} - f_{2}}\times h\), where f_m is freq of modal class, f_{1}, f_{2} are adjacent frequencies, l lower boundary and h class width.
Q1. Find mean of the data: 4, 7, 10, 6, 9.
Sum = 4+7+10+6+9 = 36. Number of observations n=5.
Mean = 36/5 = 7.2.
Q2. Find the median of the data: 12, 7, 9, 15, 10.
Sort the data: 7, 9, 10, 12, 15. n=5 (odd) so median is middle value = 10.
Q3. Find the mode of: 2, 5, 2, 7, 9, 2, 5.
Frequency counts: 2 → 3 times, 5 → 2 times, 7 →1, 9→1. Mode = value with highest frequency = 2.

Total N = 5+8+12+5 = 30. Median position = N/2 = 15th observation. Cumulative frequencies: 0–10:5, 10–20:13, 20–30:25. Median class is 20–30 (since cumulative crosses 15 there). Use formula:
\(\text{Median} = l + \dfrac{\frac{N}{2} - c_f}{f_m} \times h\), where l=20 (lower boundary), c_f = cumulative frequency before median class =13, f_m =12 (freq of median class), h=10 (class width).
Median = 20 + ((15 − 13)/12)×10 = 20 + (2/12)×10 = 20 + (1.666...) = 21.67 (approx).

Q8. Find variance and standard deviation (population) for data: 2, 4, 4, 6, 10.
Mean = (2+4+4+6+10)/5 = 26/5 = 5.2.
Compute squared deviations: (2−5.2)²=10.24, (4−5.2)²=1.44, (4−5.2)²=1.44, (6−5.2)²=0.64, (10−5.2)²=23.04. Sum = 36.8.
Population variance σ² = (Σ (x−μ)²)/N = 36.8/5 = 7.36. Standard deviation σ = √7.36 ≈ 2.713.

Class marks: 5, 15, 25. Frequencies: 5,8,7. N=20.
Mean ≈ (5·5 + 15·8 + 25·7)/20 = (25 + 120 + 175)/20 = 320/20 = 16.
Compute Σf(m−mean)²: (5(5−16)²)=5×121=605; 8(15−16)²=8×1=8; 7(25−16)²=7×81=567. Sum=1180.
Variance ≈ 1180/20 = 59. SD ≈ √59 ≈ 7.681.
Q10. Effect of outlier: Data set A: 5,6,7,8,9 and B where one value changed to 50: 5,6,7,8,50. Compare mean & median.
For A: mean = (5+6+7+8+9)/5 = 35/5 = 7; median = 7.
For B: mean = (5+6+7+8+50)/5 = 76/5 = 15.2; median = 7.
Observation: Mean is sensitive to outliers (shifted from 7 to 15.2), median is robust (stays 7). Use median for skewed data.
