Class 11 Mathematics – Chapter 2: Relations and Functions

Important Questions with Detailed Answers

Question 1

Define a relation. Give an example of a relation from set A = {1,2,3} to B = {4,5}.

Answer: A relation from set A to set B is a subset of the Cartesian product A × B.
For example, let R = {(1,4), (2,5)}. This is a relation because it is a subset of A × B = {(1,4), (1,5), (2,4), (2,5), (3,4), (3,5)}.

Question 2

If A = {1,2}, B = {x,y}, find A × B. How many relations can be defined from A to B?

Answer: A × B = {(1,x), (1,y), (2,x), (2,y)}.
The total number of relations = number of subsets of A × B = 24 = 16.

Question 3

Show that the relation R on set N given by R = {(a,b): a divides b} is reflexive, antisymmetric, and transitive.

Answer:
• Reflexive: Every natural number a divides itself, so (a,a) ∈ R.
• Antisymmetric: If a divides b and b divides a, then a = b.
• Transitive: If a divides b and b divides c, then a divides c.
Hence, R is a partial order relation.

Question 4

Determine whether the relation R on set A = {1,2,3} defined by R = {(1,1), (2,2), (3,3), (1,2), (2,3)} is transitive.

Answer: For transitivity: if (a,b) ∈ R and (b,c) ∈ R, then (a,c) must be ∈ R.
We have (1,2) and (2,3) in R, so (1,3) must be in R. But it is not. Hence, R is not transitive.

Question 5

What is a function? Give an example.

Answer: A relation is a function if every element of the domain is related to exactly one element of the codomain.
Example: f: A = {1,2,3} → B = {a,b} defined by f = {(1,a), (2,b), (3,a)} is a function because each element of A has a unique image in B.

Question 6

If f: R → R defined by f(x) = x², show that f is not one-one but is many-one.

Answer: f(2) = 4 and f(-2) = 4. Since two different inputs give the same output, f is not one-one. But multiple elements mapping to the same element means f is many-one.

Question 7

Find the domain and range of f(x) = √x.

Answer: Since square root is defined only for x ≥ 0,
Domain = [0, ∞).
Range = [0, ∞).

Question 8

Check if f: R → R, f(x) = 2x + 3 is bijective.

Answer: One-one: If f(x₁) = f(x₂), then 2x₁+3 = 2x₂+3 ⇒ x₁=x₂. So f is injective.
Onto: For any y ∈ R, take x = (y-3)/2 ∈ R such that f(x) = y. So f is surjective.
Hence, f is bijective.

Question 9

Find the range of f(x) = 1/(x-1), x ≠ 1.

Answer: The function is defined for all real x except 1.
The range is R \ {0} (all real numbers except 0), because 1/(x-1) can never be 0.

Question 10

If f(x) = x³, show that f is one-one and onto.

Answer: One-one: If f(x₁) = f(x₂) ⇒ x₁³ = x₂³ ⇒ x₁ = x₂. So injective.
Onto: For any y ∈ R, ∃ x = ³√y ∈ R such that f(x) = y. So surjective.
Hence, f is bijective.
