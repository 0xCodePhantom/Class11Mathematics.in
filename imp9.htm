Chapter 9: Straight Lines – Important Questions with Solutions

Q1. Find the slope of the line passing through points (2, 3) and (5, 11).

Solution: Slope formula: m = (y₂ - y₁)/(x₂ - x₁).

Here, (x₁, y₁) = (2, 3), (x₂, y₂) = (5, 11).

m = (11 - 3) / (5 - 2) = 8 / 3.

Answer: Slope = 8/3.

Q2. Write the equation of the line with slope 2 passing through (1, -3).

Equation: y - y₁ = m(x - x₁).

y - (-3) = 2(x - 1) → y + 3 = 2x - 2 → y = 2x - 5.

Answer: Equation = y = 2x - 5.

Q3. Find the intercepts of the line 3x + 4y = 12.

For x-intercept: put y = 0 → 3x = 12 → x = 4.

For y-intercept: put x = 0 → 4y = 12 → y = 3.

Answer: x-intercept = 4, y-intercept = 3.

Q4. Find the equation of a line passing through (2, 2) and parallel to 3x - y + 7 = 0.

Equation form: 3x - y + c = 0.

Put (2,2): 3(2) - 2 + c = 0 → 6 - 2 + c = 0 → c = -4.

Answer: Equation = 3x - y - 4 = 0.

Q5. Find the angle between lines y = √3x and y = -√3x.

Slopes: m₁ = √3, m₂ = -√3.

Formula: tanθ = |(m₁ - m₂)/(1 + m₁m₂)|.

= |(√3 - (-√3))/(1 + (√3)(-√3))| = |2√3/(1 - 3)| = |2√3 / -2| = √3.

θ = tan⁻¹(√3) = 60°.

Answer: Angle = 60°.

Q6. Determine if lines 2x - 3y + 5 = 0 and 3x + 2y - 7 = 0 are perpendicular.

Line1 slope = 2/3, Line2 slope = -3/2.

Product = (2/3)(-3/2) = -1.

Answer: Yes, lines are perpendicular.

Q7. Find the distance of point (3, -2) from line 4x + 3y + 5 = 0.

Formula: d = |ax + by + c| / √(a²+b²).

= |4(3) + 3(-2) + 5| / √(4²+3²) = |12 - 6 + 5| / 5 = 11/5.

Answer: Distance = 11/5 units.

Q8. Find equation of line passing through (0, 0) and making 45° with x-axis.

Slope = tan45° = 1.

Equation: y = x.

Answer: Equation = y = x.

Q9. The vertices of a triangle are (0, 0), (4, 0), (0, 3). Find equation of altitude from (0, 0).

Base = line through (4, 0) and (0, 3).

Equation: slope = (3 - 0)/(0 - 4) = -3/4.

Equation: y - 0 = (-3/4)(x - 4) → 3x + 4y - 12 = 0.

Altitude through (0, 0): slope = 4/3.

Equation: y = (4/3)x.

Answer: Equation of altitude = y = (4/3)x.

Q10. Find the equation of the line which bisects the angle between x-axis and y-axis.

Angle bisectors are y = ±x.

Answer: Equations = y = x or y = -x.
