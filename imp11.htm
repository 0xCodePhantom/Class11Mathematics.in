Chapter 11: Three Dimensional Geometry

Key questions, clear step-by-step solutions and diagrams (Class 11 CBSE).
Overview
Three dimensional geometry works with points (x,y,z) in space. Key objects: lines, planes, distances, direction ratios/ratios, and angles between lines/planes. Coordinate axes are x, y, z — visualize using projections.
Q1. Find the distance between points A(1,2,3) and B(4,6,3).
Solution: Distance formula in 3D: |AB| = √[(x₂−x₁)²+(y₂−y₁)²+(z₂−z₁)²].
Here: Δx=3, Δy=4, Δz=0 ⇒ |AB| = √(9+16+0) = √25 = 5.
Q2. Write symmetric and parametric equations of line passing through P(1,0,2) with direction ratios 2,−1,3.
Parametric: x=1+2t, y=0−t, z=2+3t, t∈R.
Symmetric form: (x−1)/2 = (y−0)/(−1) = (z−2)/3.
Q3. Find equation of plane through points A(1,0,0), B(0,1,0), C(0,0,1).
Two direction vectors: AB = B−A = (−1,1,0), AC = C−A = (−1,0,1). Normal vector n = AB × AC.
AB×AC = |i j k; −1 1 0; −1 0 1| = (1·1 − 0·0)i − (−1·1 − 0·(−1))j + (−1·0 − 1·(−1))k = (1)i − (−1)j + (1)k = (1,1,1). So plane eqn: 1(x−1)+1(y−0)+1(z−0)=0 ⇒ x−1 + y + z =0 ⇒ x + y + z = 1.
Q4. Find angle between lines L₁: r = (1,2,3) + λ(2,−1,1) and L₂: r = (0,1,−1) + μ(1,2,−1).
Angle between lines = angle between their direction vectors a=(2,−1,1) and b=(1,2,−1).
Use cosθ = (a·b)/(|a||b|). a·b = 2·1 + (−1)·2 + 1·(−1) = 2 −2 −1 = −1.
|a| = √(4+1+1)=√6, |b|=√(1+4+1)=√6 ⇒ cosθ = (−1)/6 ⇒ θ = cos⁻¹(−1/6) ≈ 99.59°.
Q5. Shortest distance between skew lines L₁: r = (1,0,2) + λ(1,1,0) and L₂: r = (0,1,−1) + μ(2,−1,1).
Direction vectors a=(1,1,0), b=(2,−1,1). Connect point P₁=(1,0,2) to P₂=(0,1,−1): c = P₂−P₁=(−1,1,−3).
Shortest distance = |c·(a×b)| / |a×b|. Compute a×b = |i j k; 1 1 0; 2 −1 1| = (1·1 − 0·(−1))i − (1·1 − 0·2)j + (1·(−1) − 1·2)k = (1)i − (1)j + (−3)k = (1,−1,−3).
c·(a×b) = (−1)·1 + 1·(−1) + (−3)·(−3) = −1 −1 +9 = 7. |a×b| = √(1+1+9)=√11. Distance = |7|/√11 ≈ 2.11.
Q6. Find equation of plane with normal vector n=(2,−3,1) passing through point (1,2,−1).
Plane equation: n·(r−r₀)=0 ⇒ 2(x−1) −3(y−2) +1(z+1)=0 ⇒ 2x−2 −3y+6 + z +1=0 ⇒ 2x −3y + z +5 =0.
Q7. Distance of point (2,1,3) from plane 3x − 6y + 2z − 4 = 0.
Formula: d = |ax₀+by₀+cz₀+d| / √(a²+b²+c²).
Numerator = |3·2 −6·1 +2·3 −4| = |6 −6 +6 −4| = |2| = 2. Denominator = √(9+36+4)=√49=7 → d=2/7.
Q8. Find intersection point of line r=(2,−1,3)+t(1,2,−1) with plane x + y + z = 4.
Parametric line: x=2+t, y=−1+2t, z=3−t. Substitute: (2+t) + (−1+2t) + (3−t) = 4 ⇒ 2+t −1+2t +3 − t =4 ⇒ (2+ −1+3) + (t+2t−t) =4 ⇒ 4 +2t =4 ⇒ 2t=0 ⇒ t=0. Point: (2,−1,3).
Q9. Equation of plane through three points P(1,0,0), Q(0,2,0), R(0,0,3).
Vectors PQ=(−1,2,0), PR=(−1,0,3). Normal n = PQ×PR = |i j k; −1 2 0; −1 0 3| = (2·3−0·0)i − (−1·3 −0·(−1))j + (−1·0 −2·(−1))k = (6)i − (−3)j + (2)k = (6,3,2). Take point P(1,0,0): 6(x−1)+3(y−0)+2(z−0)=0 ⇒ 6x−6 +3y +2z =0 ⇒ 6x +3y +2z =6 ⇒ divide by 1: 6x + 3y + 2z = 6.
Q10. Show projection of vector a=(3,4,0) on b=(1,2,2).
Projection scalar = (a·b)/|b|. a·b = 3·1 +4·2 +0·2 = 3 +8 =11. |b| = √(1+4+4)=√9=3. Scalar projection = 11/3. Vector projection = [(a·b)/|b|²] b = (11/9)·(1,2,2) = (11/9,22/9,22/9).
