Chapter 4: Complex Numbers and Quadratic Equations

Important Questions with Detailed Answers

1. Express √(-25) in the form a + ib.
√(-25) = √(25) × √(-1) = 5i.
So, in a + ib form: 0 + 5i.
2. Find the conjugate and modulus of z = 3 – 4i.
Conjugate: 3 + 4i.
Modulus: |z| = √(3² + (−4)²) = √(9 + 16) = √25 = 5.
3. Represent z = 3 – 4i on the Argand plane.
The point (3, −4) lies in the 4th quadrant.
4. Solve the quadratic equation x² + 4x + 5 = 0.
Discriminant D = b² − 4ac = 16 − 20 = −4 < 0.
Roots = [−4 ± √(−4)] / 2 = [−4 ± 2i] / 2 = −2 ± i.
5. If z = 1 + i, find z² and |z|.
z² = (1 + i)² = 1 + 2i + i² = 1 + 2i − 1 = 2i.
|z| = √(1² + 1²) = √2.
6. Solve x² + x + 1 = 0.
D = 1 − 4 = −3.
Roots = (−1 ± √(−3))/2 = (−1 ± i√3)/2.
7. If z = cosθ + i sinθ, show that |z| = 1.
|z| = √(cos²θ + sin²θ) = √1 = 1.
8. Represent cube roots of unity on the Argand plane.
Cube roots: 1, (−1 + i√3)/2, (−1 − i√3)/2. They form an equilateral triangle on the unit circle.
9. Express 1/(1 − i) in a + ib form.
Multiply numerator & denominator by (1 + i):
1/(1 − i) × (1 + i)/(1 + i) = (1 + i)/(1 + 1) = (1 + i)/2 = 0.5 + 0.5i.
10. Solve x² + 2x + 10 = 0.
D = 4 − 40 = −36.
Roots = (−2 ± √(−36)) / 2 = (−2 ± 6i)/2 = −1 ± 3i.
