Class 11 Mathematics – Chapter 1: Sets

Detailed answers with built-in SVG diagrams (CBSE 2025–26)

Q1.
Define a set. Give two examples from daily life and represent them in roster and set-builder form.
A set is a well-defined collection of distinct objects. "Well-defined" means there is no ambiguity about whether any particular object belongs to the set or not.

Example 1 — vowels:
Roster: A = {a, e, i, o, u}
Set-builder: A = {x | x is a vowel in the English alphabet}

Example 2 — natural numbers less than 6:
Roster: B = {1, 2, 3, 4, 5}
Set-builder: B = {x | x ∈ N and x < 6}
Q2.
Write all subsets of the set A = {1, 2, 3}. How many subsets are there in total?
Subsets are all possible selections of elements (including the empty set and the full set). List: {}, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3}.
Reason: for each element you either include it or not, so total = 2ⁿ = 2³ = 8.
Q3.
If A = {x : x is a natural number less than 6}, list the elements of A and find its power set.
A = {1, 2, 3, 4, 5}. Power set P(A) is the set of all subsets. Since A has 5 elements, P(A) has 2⁵ = 32 subsets. (You do not normally write all 32 for answers; state the count and show understanding.)
Q4.
State and verify De Morgan’s Laws for sets with an example.
De Morgan’s Laws:
(i) (A ∪ B)′ = A′ ∩ B′
(ii) (A ∩ B)′ = A′ ∪ B′

Why it makes sense (plain words): The complement of the union are those elements that are in neither A nor B — which exactly means they are in A's complement and in B's complement at the same time.
(A ∪ B)′ — elements in neither A nor B
A′ ∩ B′ — same shaded area, so equal
Both pictures shade the same region — that's the visual proof.
Q5.
If U = {1, 2, …, 10}, A = {2,4,6,8,10}, B = {1,2,3,4,5}, find A ∪ B, A ∩ B, A′, and B′.
Compute step-by-step:
A ∪ B = elements in A or B: {1,2,3,4,5,6,8,10}.
A ∩ B = common elements: {2,4}.
A′ = U \ A = {1,3,5,7,9}.
B′ = U \ B = {6,7,8,9,10}.
Q6.
Show that (A ∪ B) ∩ C = (A ∩ C) ∪ (B ∩ C) using a Venn diagram.
Idea: Shade (A ∪ B) first, then keep only the portion inside C. That will be exactly the union of A∩C and B∩C.
Shade elements that are in C and also in A or B.

Q7.
For sets A, B, and C, prove A × (B ∪ C) = (A × B) ∪ (A × C).
Explanation: A × (B ∪ C) is the set of ordered pairs (a, x) where a ∈ A and x ∈ (B ∪ C). That means x belongs to B or C. If x ∈ B then (a,x) ∈ A × B; if x ∈ C then (a,x) ∈ A × C. So every element of the left-hand side is in the right-hand side. The opposite inclusion is similar. Therefore both sets are equal.
Q8.
Find the number of elements in the power set of {a, b, c, d}.
Set has 4 elements → power set size = 2⁴ = 16 (including empty set and full set).
Q9.
Two sets A and B have n(A) = 20, n(B) = 28 and n(A ∪ B) = 36. Find n(A ∩ B).
Use the formula: n(A ∪ B) = n(A) + n(B) − n(A ∩ B).
So n(A ∩ B) = n(A) + n(B) − n(A ∪ B) = 20 + 28 − 36 = 12.
Q10.
Out of 100 students, 55 play cricket, 45 play football and 20 play both. How many play neither?
Students who play cricket or football = n(C) + n(F) − n(C ∩ F) = 55 + 45 − 20 = 80.
So neither = 100 − 80 = 20.
Numbers shown: Cricket only = 55−20 = 35, Football only = 45−20 = 25, Both = 20, Neither = 100−80 = 20.
