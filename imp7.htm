Chapter 7: Binomial Theorem – Important Questions

Q1. State the Binomial Theorem for a positive integer n.

Binomial Theorem:

(a + b)n = Σk=0n C(n,k) · an-k bk

Here, C(n,k) = n! / (k!(n-k)!)

Q2. Expand (x + y)5 using the Binomial Theorem.

(x + y)5 = C(5,0)x5 + C(5,1)x4y + C(5,2)x3y2 + C(5,3)x2y3 + C(5,4)xy4 + C(5,5)y5

= x5 + 5x4y + 10x3y2 + 10x2y3 + 5xy4 + y5

Q3. Find the middle term in the expansion of (1 + x)10.

Number of terms = n+1 = 11.

Since 11 is odd, middle term is the 6th term = T6.

T6 = C(10,5)(1)5x5 = 252x5

Q4. Find the coefficient of x7 in (2 + x)10.

General term: Tk+1 = C(10,k)(2)10-kxk

We need k=7: T8 = C(10,7)(2)3x7 = 120 × 8 × x7 = 960x7

Coefficient: 960

Q5. Expand (1 + x)6 and find the sum of coefficients.

Expansion: (1 + x)6 = Σ C(6,k)xk

Sum of coefficients = put x = 1 = (1+1)6 = 64

Q6. Find the constant term in (x + 1/x)8.

General term: Tk+1 = C(8,k)x8-k(1/x)k = C(8,k)x8-2k

For constant term, power of x = 0 ⇒ 8 – 2k = 0 ⇒ k = 4

Constant term = C(8,4) = 70

Q7. Prove that the sum of coefficients in (1 – x)n is 0 if n is odd.

Sum of coefficients = put x = 1

(1 – 1)n = 0, if n is odd ⇒ sum = 0

Q8. Find the coefficient of x9 in (1 + x)18.

General term: Tk+1 = C(18,k)xk

We want power 9 ⇒ k=9

Coefficient = C(18,9) = 48620

Q9. Show that (1 + x)2n contains term independent of x when n is even.

General term: Tk+1 = C(2n,k)xk

Constant term requires k = n

Coefficient = C(2n,n). Exists only if n is integer, so term is constant when n is even.

Q10. Expand (1 – 2x)4.

(1 – 2x)4 = Σ C(4,k)(1)4-k(–2x)k

= 1 – 8x + 24x2 – 32x3 + 16x4
